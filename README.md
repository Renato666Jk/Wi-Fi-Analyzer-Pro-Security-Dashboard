# 📡 Wi-Fi Analyzer Pro

O **Wi-Fi Analyzer Pro** é uma ferramenta de auditoria de rede em Python (Flask) com interface moderna. Ele oferece um **Radar Sonar** para sinal, **Scanner de Câmeras LAN** para privacidade e **Auditoria de Segurança** (WPA2/WEP). Identifica fabricantes via API e exporta relatórios CSV. Foco total em segurança e performance! 📡🚀

## ✨ Funcionalidades
- **Visualização Sonar:** Interface animada para proximidade de redes.
- **Scanner de Dispositivos:** Identifica câmeras IP e vulnerabilidades na LAN.
- **Auditoria de Segurança:** Classifica riscos de protocolos Wi-Fi.
- **Relatórios:** Exportação de dados técnicos para CSV.

## 🛠️ Tecnologias
- **Backend:** Python & Flask
- **Frontend:** HTML5, CSS3 (Vibe Coding Style) & Chart.js
- **Network:** PyWiFi & Scapy
